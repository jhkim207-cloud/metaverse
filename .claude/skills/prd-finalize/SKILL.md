# /prd-finalize - PRD 통합 Skill

## 개요

모든 설계 문서를 통합 PRD로 병합하고, 전체 검증 게이트를 실행하여 80% 완성도를 판정합니다.

## 사용법

```
/prd-finalize [기능명]
```

## 입출력

- **입력**: `prd/{기능명}/00~06-*.md` (7개 문서)
- **산출물**: `prd/{기능명}-prd.md`

---

## 🔴 필수 실행 체크리스트

실행 전 반드시 확인:

```
□ 1. Gate 0~6 통과 여부 집계
□ 2. Provisional 항목 수집 (컴포넌트/트렌드/용어)
□ 3. 사용자에게 Provisional 승인 요청
□ 4. 승인된 항목 → 표준 문서 업데이트
□ 5. 80% 완성도 판정 (46개 중 37개+)
□ 6. PRD 통합 문서 생성
```
