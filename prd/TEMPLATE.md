## 5. API 설계

### 5.1 API 목록

| Method | Endpoint | 설명 | 인증 |
|--------|----------|------|------|
| GET | /api/v1/... | | 필요 |
| POST | /api/v1/... | | 필요 |
| PUT | /api/v1/... | | 필요 |
| DELETE | /api/v1/... | | 필요 |

### 5.2 API 상세

#### GET /api/v1/resource

**Request:**
```
Query Parameters:
- page: int (default: 0)
- size: int (default: 20)
- sort: string (예: "createdAt,desc")
```

**Response (200 OK):**
```json
{
  "success": true,
  "data": {
    "content": [],
    "page": 0,
    "size": 20,
    "totalElements": 100,
    "totalPages": 5
  }
}
```

**Error Response (4xx/5xx):**
```json
{
  "success": false,
  "error": {
    "code": "ERR_001",
    "message": "에러 메시지"
  }
}
```
